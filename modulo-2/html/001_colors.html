Cores

Representação por Nomes

            
                h1 {
                    background-color: blue;
                    color: white;
                }

Representação Hexadecimal

            
                h1 {
                    background-color: #0000ff;
                    color: #ffffff;
                }

Representação RGB

            
                h1 {
                    background-color: rgb(0, 0, 255);
                    color: rgb(255, 255, 255);
                }

Padrão de Cor HSL

            
                h1 {
                    hsl(240, 100%, 50%); 
                    color: hsl(0, 0%, 100%);
                }

Opacidade

            
                h1 {
                    background-color: rgba(0, 0, 255, 0.485);
                    color: rgb(255, 255, 255);
                }
